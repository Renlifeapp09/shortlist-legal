# The Shortlist — Legal Documents

Privacy Policy and Terms of Service hosted via GitHub Pages.

## Setup

1. Create a new repo on GitHub called `shortlist-legal` (public)
2. Clone it locally and copy these files into it
3. Paste your Termly HTML content into `privacy.html` and `terms.html` (replace the comment blocks inside `<main>`)
4. Push to GitHub
5. Go to repo **Settings → Pages → Source** → select `main` branch, `/ (root)`, click Save
6. Your site will be live at `https://<your-github-username>.github.io/shortlist-legal/`

## URLs for App Store Connect

- Privacy Policy: `https://<your-github-username>.github.io/shortlist-legal/privacy.html`
- Terms of Service: `https://<your-github-username>.github.io/shortlist-legal/terms.html`

## Custom Domain (optional)

To use `legal.theshortlistapp.com`:

1. In repo Settings → Pages → Custom domain, enter `legal.theshortlistapp.com`
2. Add a CNAME record in your DNS: `legal` → `<your-github-username>.github.io`
3. Wait for DNS propagation (up to 24 hours)
4. Check "Enforce HTTPS" once the certificate is provisioned
